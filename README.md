🚀 CSS Generator com IA (Criado em aprendizagem durante o evento de Programação com a <b>Dev Club</b>

Uma ferramenta web interativa que transforma descrições em linguagem natural em código CSS funcional — com preview em tempo real.

💡 Basta descrever o que você imagina (ex: "estrela de cinco pontas girando") e a aplicação gera automaticamente o estilo correspondente.

🧠 Sobre o projeto

O CSS Generator com IA foi desenvolvido com o objetivo de facilitar a criação de estilos visuais, especialmente para quem está aprendendo CSS ou deseja acelerar o desenvolvimento de interfaces.

A aplicação interpreta descrições de texto e gera dinamicamente código CSS, incluindo formas complexas e animações.

---

⚙️ Funcionalidades
🔤 Geração de CSS a partir de texto
👀 Preview em tempo real
🎨 Criação de formas com clip-path
🔄 Animações com @keyframes
💻 Interface moderna e responsiva
